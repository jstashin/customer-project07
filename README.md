
 - What is JSX, and why is it used?

JSX is a syntax extension that is used in React.The code is looked like HTML,but it is written inside JavaScript.

 - What is the difference between State and Props?
State is local data storage which is managed by the component.
Props is the process to send data from parent to child components.
State is controlled by itself when Props is controlled by the parent component.

 - What is the useState hook, and how does it work?
useState is a React Hook that helps to add state to functional components.
useState returns an array which contains two items.One is the current state value and
other is a function to update that value.

 - How can you share state between components in React?
Using context API,lifting state up,using state management liabraries.
